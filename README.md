# homebrew-iflow

Homebrew tap for [iFlow CLI](https://github.com/iflow-ai/iflow-cli) — AI-powered CLI that embeds in your terminal for coding tasks and workflow automation.

## Install

```bash
brew tap gandli/iflow
brew install iflow-cli
```

## Upgrade

```bash
brew upgrade iflow-cli
```

## Uninstall

```bash
brew uninstall iflow-cli
brew untap gandli/iflow
```
